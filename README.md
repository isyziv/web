## HTML

div		插入圖片

ui ul	項目符號

p		段落

img	插入圖片

h		套用段字體大小

span	容器包裹文字方便CSS設定

nav		導航連結區塊

footer	結尾/頁尾區塊

## CSS
color、background-color 設定顏色
weight height 設定寬&高度
padding 行距設定
font- 改變字形設定
text-align 控制行內內容的對齊
display:block 元素的顯示型態被設定為區塊元素
display:inline 元素的顯示型態被設定為內行元素
